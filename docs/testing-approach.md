# Testing Approach

## Unit Tests
- URL validation
- Custom alias uniqueness
- Short code generation
- Expiry and inactive link logic

## Integration Tests
- Create URL endpoint
- Get URL endpoint
- Redirect endpoint
- Update URL endpoint
- Analytics endpoint

## Manual Testing
- Submit invalid URL values
- Create and retrieve short URLs
- Test duplicate alias rejection
- Test redirect behavior for expired and inactive links
- Confirm analytics incrementation after redirect
